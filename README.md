1. Project Title / Headline

E-Commerce Sales Dashboard using Power BI

2. Short Description / Purpose

Designed and developed an interactive Power BI dashboard to analyze and visualize e-commerce sales performance across states, customers, and product categories. The goal was to identify top-performing regions, profitable categories, and customer trends for better business insights and decision-making.

3. Tech Stack

Power BI Desktop – for data modeling and dashboard creation
Microsoft Excel / CSV – for data storage and preprocessing
DAX (Data Analysis Expressions) – for calculated measures and KPIs
Power Query – for data cleaning and transformation

4. Data Source

Orders.csv – contained order-level details such as sales, profit, quantity, and category.
Details.csv – included customer, payment mode, and regional data.

5. Features / Highlights

Interactive filters by state, customer, category, and quarter
Visuals for sales, profit, and quantity analysis
Comparison of profit by month and sub-category
Insights on top-performing states and customers
Donut charts showing category and payment mode distribution
Attractive dark-themed UI with clean, intuitive visuals
